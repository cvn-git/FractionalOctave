# FractionalOctave
Python package to do spectral analysis using fractional-octave filterbanks

## Installation
### For normal use
Run the following command to get the standard installation of Fractional Octave into your Python virtual environment:
```shell
pip install git+https://github.com/cvn-git/FractionalOctave.git
```

### For FractionalOctave development
Download FractionalOctave source code from https://github.com/cvn-git/FractionalOctave and then run the following commands to install the module in editable mode together with extra dependencies: 
```shell
cd <FRACTIONAL_OCTAVE_SOURCE_PATH>
pip install -e .[development]
```
